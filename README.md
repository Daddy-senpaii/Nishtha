# Nishtha – CLI Task Manager (Go)

**Nishtha** is a lightweight command-line task manager built with **Go**.  
It helps manage and schedule tasks directly from the terminal while demonstrating **Go generics, pointer receivers, JSON handling, and file persistence**.

---

## ✨ Features

- Add, update, delete, and list tasks
- Detect **due tasks automatically**
- Provide **notifications for upcoming tasks**
- Persistent storage using **JSON files**
- Clean and modular Go code structure

---

## 🛠 Technologies

- Go (Golang)
- JSON Encoding / Decoding
- File I/O
- Go Generics
- Pointer Receivers
